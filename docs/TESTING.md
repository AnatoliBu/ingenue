# Testing

Run the transport diagnostics unit tests with:

```bash
npm test
```

On a norns device with this branch installed, open:

```text
http://norns.local:7777/transport-diagnostics.html
```

Run the default 100-sample test first while idle, then repeat while a representative script is playing. Copy the JSON report into issue #2 with the device, browser, network mode, and script name.
