⚠ install from an untrusted source?

norns scripts run with full access to your device. only install scripts you trust. ingenue's maintainer doesn't review, vouch for, or take responsibility for third-party scripts — that responsibility is yours.
⚙ engine name conflict

SuperCollider registers each engine by class name. A duplicate makes one engine fail to load — and it's invisible until a script runs.

If this script ships the same engine, reuse the installed one — no duplicate (verified identical on device before linking).

this script needs setup

Setup runs the script's own installer (install.sh) and fetches the downloads above. It can take several minutes and runs commands on your device.
SuperCollider plugins

After installing, you must fully reboot the whole device (not just restart the norns app) — SuperCollider only loads plugin binaries at boot.
