# screenshots

Drop the captured images in here (referenced by the top-level `README.md` gallery).
The README nests them in `<details>` dropdowns (Desktop / Phone).

## desktop/ (PNG)
- `editor.png`       — editor + live REPL (also the hero image)
- `file-browser.png` — dust file browser (select / upload / download / mkdir / rename / delete)
- `params.png`       — live PARAMS: nested groups, presets, PSET, note-name sliders
- `repo.png`         — repository manager: tag filter + cards + embedded demos
- `discover.png`     — GitHub discovery with facet badges (`script` / `mod` / `library` / `engine`)
- `mods.png`         — mods sheet (installed toggles + installable community mods)

## phone/ (JPG)
- `editor.jpg`       — editor (and the in-app update toast)
- `file-browser.jpg` — files tab
- `matron.jpg`       — live REPL
- `params.jpg`       — PARAMS
- `repo.jpg`         — scripts: installed / available / discover
- `themes.jpg`       — configuration + theme picker (137 schemes)

Tip: resize a desktop browser narrow (≤ 640px) to grab the phone shots — the layout
is driven by container-query width, so it reflows to the mobile UI automatically.
