# screenshots

Drop the captured PNGs in here (referenced by the top-level `README.md` gallery).
The README nests them in `<details>` dropdowns (Desktop / Phone).

## desktop/
- `editor.png`        — editor + live REPL (also the hero image)
- `params.png`        — live PARAMS: nested groups, dropdowns, note-name sliders
- `repo.png`          — repository manager with the tag-filter facet + cards
- `repo-carousel.png` — an expanded card showing the README image carousel ("images flipper")
- `repo-demo.png`     — an expanded card with an embedded video demo
- `mods.png`          — mods sheet (installed toggles + installable community mods)
- `themes.png`        — config / theme picker (137 schemes)

## phone/
- `repo.png` · `params.png` · `editor.png` — the same views at phone width

Tip: resize a desktop browser narrow (≤ 640px) to grab the phone shots — the layout
is driven by container-query width, so it reflows to the mobile UI automatically.
